# TapXBot - Tap & Earn X

A fun Telegram bot game where users earn XP by tapping a lion head and completing various tasks and missions.

## 🎮 Features

- **Tap to Earn**: Click the lion to earn XP
- **Lion Head SVG**: Beautiful, interactive lion with golden ornament ring
- **Daily Rewards**: Complete daily tasks for bonus XP
- **Missions**: Various missions to earn extra rewards
- **Bounties**: Special bounties and leaderboards
- **Wallet**: Manage earnings and withdraw XP
- **Referral System**: Invite friends and earn rewards
- **Multiple Ad Types**: Watch ads for bonus rewards

## 📁 Pages

1. **index.html** - Main page with lion tapping and core features
2. **missions.html** - Daily and weekly missions
3. **bounty.html** - Special bounties and leaderboards
4. **wallet.html** - Withdrawal and transaction history

## 🚀 Getting Started

Simply open `index.html` in your browser or deploy to a web server.

### Integration with Telegram

Add this bot to your Telegram bot using the Telegram Bot API:

```
https://t.me/TapXBot
```

- **Floating Text**: +XP text animation on tap
- **Progress Bars**: Visual progression tracking

## 📊 Game Mechanics

- **Taps**: +1 XP per tap
- **Ad Rewards**: +10 XP for watching ads
- **Task Rewards**: +10 XP per completed task
- **Daily Bonus**: +900 XP after completing 4 daily tasks
- **Referral**: +5 XP per friend join

## 💰 Ranks

- Starter: 0-499 XP
- Novice: 500-999 XP
- Elite: 1000-1999 XP
- Champion: 2000-4999 XP
- Legend: 5000+ XP
